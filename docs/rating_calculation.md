# レーティングの計算方法

本ドキュメントでは、CHUNITHM におけるレーティング（Rating）の計算方法について解説します。

---

## 概要

レーティングは、プレイヤーの実力を数値化した指標です。以下の2つの要素から構成されます。

- **ベスト枠（Best）**: 全プレイ楽曲の中から単曲レーティング上位 **30曲** を選出
- **新曲枠（New）**: 新曲の中から単曲レーティング上位 **20曲** を選出

プレイヤーレーティングは、この合計50曲を基に算出されます。

---

## 1. 単曲レーティングの計算

単曲レーティングは、**スコア**（0 ～ 1,010,000）と**譜面定数**の2つの値から算出されます。

### 計算式（スコアランク別）

| ランク | スコア範囲 | 計算式 |
|---|---|---|
| SSS+ | 1,009,000 ～ | 譜面定数 + 2.15 |
| SSS | 1,007,500 ～ 1,008,999 | 譜面定数 + 2.0 + (スコア − 1,007,500) ÷ 100 × 0.01 |
| SS+ | 1,005,000 ～ 1,007,499 | 譜面定数 + 1.5 + (スコア − 1,005,000) ÷ 50 × 0.01 |
| SS | 1,000,000 ～ 1,004,999 | 譜面定数 + 1.0 + (スコア − 1,000,000) ÷ 100 × 0.01 |
| S+ | 990,000 ～ 999,999 | 譜面定数 + 0.6 + (スコア − 990,000) ÷ 250 × 0.01 |
| S | 975,000 ～ 989,999 | 譜面定数 + (スコア − 975,000) ÷ 2,500 × 0.1 |
| AAA | 950,000 ～ 974,999 | 譜面定数 − 1.67 + (スコア − 950,000) ÷ 150 × 0.01 |
| AA | 925,000 ～ 949,999 | 譜面定数 − 3.34 + (スコア − 925,000) ÷ 150 × 0.01 |
| A | 900,000 ～ 924,999 | 譜面定数 − 5.0 + (スコア − 900,000) ÷ 150 × 0.01 |
| BBB | 800,000 ～ 899,999 | (譜面定数 − 5.0) ÷ 2 + (スコア − 800,000) ÷ (2,000 ÷ (譜面定数 − 5)) × 0.01 |
| C | 500,000 ～ 799,999 | (スコア − 500,000) ÷ (6,000 ÷ (譜面定数 − 5)) × 0.01 |
| D | ～ 499,999 | 0 |

### 精度

- 計算結果は小数点以下2桁で**切り捨て**
- 結果が負の値になる場合は **0**

> **注意**: 譜面定数が5.0以下の場合、BBB・Cランクの計算では0として扱います。

### 計算例

| スコア | 譜面定数 | 単曲レーティング |
|---|---|---|
| 1,010,000（理論値） | 15.0 | 17.15 |
| 1,009,000（SSS+ボーダー） | 15.0 | 17.15 |
| 1,007,500（SSSボーダー） | 15.0 | 17.00 |
| 1,005,000（SS+ボーダー） | 15.0 | 16.50 |
| 1,000,000（SSボーダー） | 15.0 | 16.00 |
| 990,000（S+ボーダー） | 15.0 | 15.60 |
| 975,000（Sボーダー） | 15.0 | 15.00 |
| 950,000（AAAボーダー） | 15.0 | 13.33 |
| 925,000（AAボーダー） | 15.0 | 11.66 |
| 900,000（Aボーダー） | 15.0 | 10.00 |
| 800,000（BBBボーダー） | 15.0 | 5.00 |
| 500,000（Cボーダー） | 15.0 | 0.00 |

## 2. プレイヤーレーティングの計算

### 構成

プレイヤーレーティングは以下の手順で計算されます。

1. **全プレイ記録の単曲レーティングを計算**する
2. **ベスト枠**: 全記録をレーティング降順にソートし、上位 **30曲** を選出
3. **新曲枠**: 新曲のみ抽出し、レーティング降順で上位 **20曲** を選出
4. **プレイヤーレーティング**を算出

### 計算式

$$
\text{プレイヤーレーティング} = \text{trunc}\left(\frac{\text{ベスト枠合計} + \text{新曲枠合計}}{50}, 2\right)
$$

$$
\text{ベスト枠平均} = \text{trunc}\left(\frac{\text{ベスト枠合計}}{\text{ベスト枠曲数}}, 2\right)
$$

$$
\text{新曲枠平均} = \text{trunc}\left(\frac{\text{新曲枠合計}}{\text{新曲枠曲数}}, 2\right)
$$

ここで $\text{trunc}(x, n)$ は小数点以下 $n$ 桁で切り捨てを意味します。

### 重要な仕様

- **除数は常に50**: ベスト枠30曲 + 新曲枠20曲 = 50曲に満たない場合でも、除数は **50固定** です。つまり、プレイ曲数が少ないほどプレイヤーレーティングは低くなります。
- **新曲の重複カウント**: 新曲のレコードはベスト枠にも選出されるため、優秀な新曲スコアは2重に加算されます。

### 計算例

#### 例1: ベスト枠のみ30曲（全て旧曲）

```
各曲: SSS+（1,009,000点）、譜面定数 15.0 ～ 12.1（0.1刻み）

ベスト枠合計: 17.15 + 17.05 + ... + 14.25 = 471.0
ベスト枠平均: 471.0 ÷ 30 = 15.70
新曲枠合計:   0
新曲枠平均:   0

プレイヤーレーティング: (471.0 + 0) ÷ 50 = 9.42
```

#### 例2: 新曲枠のみ20曲（全て新曲）

```
各曲: SSS+（1,009,000点）、譜面定数 15.0 ～ 13.1（0.1刻み）

ベスト枠: 新曲20曲がそのまま選出（30枠中20曲のみ）
ベスト枠合計: 17.15 + 17.05 + ... + 15.25 = 324.0
ベスト枠平均: 324.0 ÷ 20 = 16.20
新曲枠合計:   324.0
新曲枠平均:   324.0 ÷ 20 = 16.20

プレイヤーレーティング: (324.0 + 324.0) ÷ 50 = 12.96
```
